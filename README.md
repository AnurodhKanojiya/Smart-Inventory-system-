

# Inventory Management System (Java + PostgreSQL)

This is a simple **Inventory Management System** built using **Core Java**, **JDBC**, and **PostgreSQL**.  
It allows you to manage products, update quantities, search items, and perform CRUD operations.

 Features

- Add new products  
- Search product by SKU or ID  
- Update product quantity  
- List all products  
- Database connection using JDBC  
- Clean DAO structure (ProductDao)
